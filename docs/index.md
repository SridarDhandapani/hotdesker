# Hotdesker

Bulk-book hot desks for the week ahead, in one click. Built for WeWork's member portal.

[Install from the Chrome Web Store →](https://chromewebstore.google.com/detail/hotdesker/hobjmhjbmhgocoffijifcenldiciclpc)

[Source on GitHub →](https://github.com/SridarDhandapani/hotdesker)

[Privacy Policy →](privacy.html)

## What it does

Tap days on a calendar, pick a location for each — different locations on different days are fine — and click Start. Hotdesker books them sequentially with a live progress bar. Cancel a single booking by tapping its day, or bulk-cancel several at once.

## How it works

The extension uses the same APIs your WeWork tab uses. Your session, your data, your machine — nothing leaves it.

See the [README](https://github.com/SridarDhandapani/hotdesker#readme) for the full feature list.
